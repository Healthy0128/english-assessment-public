# 英語音読評価ツール (公開版)

## 概要

このプロジェクトは、生徒が自分のペースで英語の音読練習を行い、AIによる即時フィードバックを受け取ることができるWebベースのツールです。特に、Microsoft Reading Progressのような高機能ツールが利用できない環境でも、同様の学習体験を提供することを目指して開発されました。

## 特徴

- **AIによる音読評価**: SpeechSuper APIを利用して、発音、流暢さ、リズム、速度などを詳細に評価します。
- **即時フィードバック**: 音読後すぐに評価結果が表示されるため、生徒は自身のパフォーマンスを即座に確認し、改善に繋げることができます。
- **繰り返し練習**: 何度でも音読に挑戦し、最高スコアを目指すことで、学習意欲を維持します。
- **シンプルなUI**: 直感的で使いやすいインターフェースで、生徒が迷うことなく学習に集中できます。

## セットアップ方法

このツールをローカル環境で実行するには、以下の手順に従ってください。

### 1. リポジトリをクローンする

```bash
git clone https://github.com/Healthy0128/english-assessment-public.git
cd english-assessment-public
```

### 2. 依存関係をインストールする

Node.jsとnpmがインストールされていることを確認し、プロジェクトの依存関係をインストールします。

```bash
npm install
```

### 3. 環境変数を設定する

SpeechSuper APIを利用するために、APIキーとシークレットキーを設定する必要があります。

1.  SpeechSuperのウェブサイトでアカウントを作成し、APIキー（App Key）とシークレットキー（Secret Key）を取得してください。
2.  プロジェクトのルートディレクトリに `.env` という名前のファイルを作成し、以下の内容を記述します。

    ```
    SPEECHSUPER_APP_KEY="あなたのSpeechSuper App Key"
    SPEECHSUPER_SECRET_KEY="あなたのSpeechSuper Secret Key"
    PORT=3000
    ```

    `.env.example` ファイルを参考にしてください。

### 4. サーバーを起動する

```bash
npm start
```

サーバーが起動したら、ウェブブラウザで `http://localhost:3000` にアクセスしてください。

## 使い方

1.  ブラウザでツールにアクセスします。
2.  音読したい英文を入力または選択します。
3.  「録音開始」ボタンをクリックし、英文を音読します。
4.  「録音停止」ボタンをクリックすると、AIによる評価が開始されます。
5.  評価結果（全体スコア、流暢さ、発音、リズム、速度など）が表示されます。
6.  結果を確認し、必要に応じて再挑戦します。
